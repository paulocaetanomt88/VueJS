Diretiva V-HTML - Injetando elementos HTML

A diretiva v-html é uma diretiva Vue.js usada para atualizar o innerHTML de um elemento com nossos dados.
Isso é o que o separa do v-text, o que significa que enquanto o v-text aceita string e o trata como uma string,
ele aceitará a string e o renderizará em HTML.

A diretiva v-text é uma diretiva Vue.js usada para atualizar o textContent de um elemento com nossos dados. É apenas uma boa alternativa para a sintaxe do Mustache.

* Ambas as diretivas não somente inserem, mas substituem todo o conteúdo existente do elemento html em que foi declarado.

* Por ter a característica de injetar e renderizar elementos HTML, é necessário tomar cuidado com informações que não são confiáveis
para evitar potenciais ataques.
